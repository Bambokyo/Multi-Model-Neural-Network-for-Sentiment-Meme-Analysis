# Multi-Model-Neural-Network-for-Sentiment-Meme-Analysis

Given a meme image, the model predicts the emotion the meme signifies:
## Humour:
1. hilarious 
2. not_funny
3. very_funny
4. funny

## Sarcasm:
general
not_sarcastic
twisted_meaning
sarcastic

## Offensive:
offensive
not_offensive
very_offensive
slight

## Motivational:
motivational
not_motivational
